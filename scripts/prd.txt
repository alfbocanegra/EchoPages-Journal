### 🛠️ Recommended Stack
- **Containerization**: Podman (rootless)
- **Shell Layer**: Distrobox with customizable Linux distro
- **Environment Definition**: `devcontainer.json`
- **Editor Support**: VS Code (Remote - Containers) or CLI

### ✅ Development Principles

| Feature         | Achieved by                                      |
|-----------------|--------------------------------------------------|
| Isolation       | Podman containers (no Docker daemon needed)      |
| Reproducibility | `devcontainer.json` and Git versioning           |
| Flexibility     | Distrobox with choice of Linux distributions     |
| Safety          | Rootless runtime with restricted filesystem access|
| Disk Efficiency | Layered container images reused and space-saving |

### 🧪 Setup Walkthrough Summary
1. **Install Podman** and create rootless Linux VM.
2. **Install Distrobox** and set up a development shell.
3. **Configure Dev Shell** with required packages and SDKs.
4. **Create `devcontainer.json`** to define and share setup.
5. **Bootstrap** configuration with scripts and maintain minimal host pollution.
