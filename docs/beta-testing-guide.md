# EchoPages Journal Beta Testing Guide

Thank you for participating in the EchoPages Journal beta! Your feedback is essential to help us improve the app before public launch.

## 1. Joining the Beta
- **iOS (TestFlight):**
  1. Install the TestFlight app from the App Store.
  2. Accept the invitation link sent to your email.
  3. Download and install EchoPages Journal via TestFlight.
- **Android (Google Play Beta):**
  1. Accept the beta invitation link sent to your email or join via the Play Store beta opt-in page.
  2. Download/update EchoPages Journal from the Play Store.
- **Expo (Early Access):**
  1. Install the Expo Go app from the App Store or Play Store.
  2. Scan the QR code or use the link provided by the team to open the latest build.

## 2. What to Test
- Account creation and login (Google, Apple, Dropbox)
- Creating, editing, and deleting journal entries
- Syncing data across devices
- Handwriting input (drawing, undo/redo, export)
- Media attachments (images, audio, video)
- Theming, accessibility, and font size options
- App performance and stability
- Any crashes, errors, or unexpected behavior

## 3. How to Report Bugs & Feedback
- **In-App Feedback:** Use the feedback button (if available) in the app settings.
- **Email:** Send feedback, bug reports, and screenshots to [beta@echopages.app]
- **Discord/Slack:** Join our tester community (invite link provided separately).
- Please include:
  - Device model and OS version
  - App version/build number
  - Steps to reproduce the issue
  - Screenshots or screen recordings if possible
